##
KBulletin is designed to allow you to read the latest headlines without shifting attention away from the workspace.
- Features a panel and desktop mode
- Card based scrollable layout that allows you to quickly scan content
- Expandable cards that display actual content when clicked
- Add & remove sources to better organize and curate your feed
- Bookmark system allowing you to store cards for later reading
- Filter sources by `Topic` and articles by `Word Bans` 
- Search function to find specific articles
##

### Installation
1) Select "Add or Manage Widgets"                                    <br>
       - Right-Click the taskbar                                     <br>
       - Right-Click the desktop and enter Edit Mode
       
2) Click "â¤“ Get New Widgets" and "Download New Plasma Widgets"

3) Use the search bar to search for "KBulletin" and click "Install"

4) Go back to "Add or Manage Widgets" and find KBulletin                                             <br>
       - Drag the widget to the desktop for desktop mode, resize it in Edit Mode                     <br>
       - Drag the widget to the taskbar for panel mode, open it with a click and resize the corners  <br>
   
         

### Requirements
- A Linux system with an up-to-date KDE Plasma desktop environment
- Minimum 35 years of work experience in media studies
- Driver License equivalent to Republic of Ireland Category B
