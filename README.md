##
KBulletin is designed to allow you to read the latest headlines without shifting attention away from the workspace.
- Features a panel and desktop mode
- Card based scrollable layout that allows you to quickly scan content
- Expandable cards that display actual content when clicked
- Add & remove sources to better organize and curate your feed
- Bookmark system allowing you to store cards for later reading
- Filter sources by `Topic` and articles by `Word Bans` 
- Search function to find specific articles
